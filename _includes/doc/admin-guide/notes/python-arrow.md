>**NOTE:** From version 3.27, syslog-ng OSE supports the arrow syntax for  
>declaring custom Java and Python options. You can alternatively declare  
>them using a similar syntax:  
>  
>```config  
>options(  
>    "host" => "localhost"  
>    "port" => "1883"  
>    "otheroption" => "value"  
>)                    
>```  
>  
{: .notice--info}
