## Location

/ patterndb

## Description

The container element of the pattern database.

## Attributes

- *version*: The schema version of the pattern database. The current
    version is 4.

- *pubdate*: The publication date of the XML file.

## Children

- *ruleset*

### Example

```xml
<patterndb version='4' pub_date='2010-10-25'>
```
